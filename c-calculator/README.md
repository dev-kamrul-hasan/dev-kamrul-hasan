# C-Calc-Evolution

A simple, functional calculator built in C. This project serves as a practical way for me to learn the fundamentals of C programming, input handling, and logic flow.

## 🚀 Features
- Performs basic arithmetic: Addition, Subtraction, Multiplication, and Division.
- Error handling for invalid operators.
- Retry logic that allows the user to correct mistakes without restarting the program.

## 🛠️ Logic Breakdown
This version of the calculator uses a `switch` statement to handle different mathematical operations. 
One specific challenge I solved was managing the **input buffer**:
- I used `scanf("%c", &enterIsPressed);` to consume the "Enter" key character. 
- This prevents the program from skipping inputs when it jumps back to the start after an error.
